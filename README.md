# url-shortener

## Getting Started

### Clone

```zsh
git clone https://github.com/poiu694/url-shortener
cd url-shortener
```

### Install && Start

```zsh
cd server && npm install && npm dev &
cd ../web && npm install && npm dev &
```

Open 'http://localhost:5173'
